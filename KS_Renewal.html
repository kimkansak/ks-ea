HOW TO RENEW
วิธีต่ออายุ Key

ง่ายมากครับ ทำได้ภายใน 5 นาที

ขั้นตอนต่ออายุ License Key
1
ทักหา KS Trader ผ่าน Line
แจ้งว่าต้องการต่ออายุ พร้อมบอก Account Number และแพ็กเกจที่ต้องการ
2
ชำระเงินตามแพ็กเกจ
โอนเงินตามจำนวน แล้วส่งสลิปให้ KS Trader ยืนยัน
3
รับ Key ใหม่ภายใน 15 นาที
KS Trader จะอัปเดต Key ใหม่ให้ทันที ไม่ต้องลง EA ใหม่ ไม่ต้องตั้งค่าใหม่ EA ทำงานต่อได้เลย
4
ใส่ Key ใหม่ใน EA
Remove EA → Load ใหม่ → ใส่ Key ใหม่ → OK → พร้อมเทรดต่อเลยครับ
ต่ออายุก่อน Key หมด = ไม่ต้องหยุดเทรดแม้แต่วันเดียว แนะนำต่ออายุล่วงหน้า 3 วันก่อน Key หมดเสมอครับ
