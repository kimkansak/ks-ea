ภาพรวม ระบบ

SECTION 02

สิ่งที่ ต้องมี

ก่อนเริ่มใช้งาน ตรวจสอบให้แน่ใจว่ามีครบทุกอย่างดังนี้:

  • คอมพิวเตอร์ ที่เปิดทิ้งไว้ได้ตลอดเวลาตลาดเปิด (หรือ VPS)
  • MetaTrader 5 (MT5) ติดตั้งเรียบร้อย เชื่อมต่อ Broker แล้ว
  • บัญชีเซน (Cent Account) กับโบรกเกอร์ที่มี XAUUSDm หรือ XAUUSDc
  • ทุนขั้นต่ำ 10,000 เซน (เทียบเท่า 100 USD) ในบัญชี
  • License Key ที่ได้รับจาก KS Trader Official
  • ไฟล์ EA — KS_ULTIMATE_DATA.mq5
  • ไฟล์ Indicator — SuperTrend_V5_Fixed.mq5
  • Internet เชื่อมต่อตลอดเวลาที่ EA รันอยู่
  • อนุญาต WebRequest ใน MT5 สำหรับ URL ของระบบ
สำคัญ — VPS แนะนำ

ถ้าคอมที่บ้านปิดหรือไฟดับ EA จะหยุดทำงานทันที แนะนำใช้ VPS (Virtual Private Server) เพื่อให้ระบบทำงานต่อเนื่อง 24/7 โดยไม่พึ่งคอมที่บ้านครับ

SECTION 03

ขั้นตอน ติดตั้ง

  1. 01
    ติดตั้งไฟล์ Indicator
    รับไฟล์ SuperTrend_V5_Fixed.mq5 จาก KS Trader → ลากไฟล์ไปวางในโฟลเดอร์ MQL5/Indicators/ ได้เลย ไม่ต้องทำอะไรเพิ่ม
  2. 02
    ติดตั้งไฟล์ EA
    รับไฟล์ KS_ULTIMATE_DATA.mq5 จาก KS Trader → ลากไฟล์ไปวางในโฟลเดอร์ MQL5/Experts/ ได้เลย ไม่ต้องทำอะไรเพิ่ม
  3. 03
    อนุญาต WebRequest ใน MT5
    ไปที่ Tools → Options → Expert Advisors → ติ๊ก "Allow WebRequests" แล้วเพิ่ม URL ของระบบ KS
  4. 04
    เปิด Chart XAUUSDm (หรือ XAUUSDc) Timeframe M5
    ใน MT5 คลิกขวาที่ Symbol → เปิด Chart → เปลี่ยน TF เป็น M5
  5. 05
    ลาก EA ลง Chart
    จาก Navigator panel ลาก EA ลง Chart → ใส่ License Key → ตั้งค่าตามต้องการ → กด OK
  6. 06
    ตรวจสอบ Experts Tab
    ดูที่แถบ Experts ด้านล่าง MT5 ควรเห็น KS AUTH: [AUTHORIZED] แสดงว่าพร้อมใช้งาน
ห้ามลาก EA สองตัวพร้อมกัน

ห้ามลาก EA ลงบน Chart เดียวกันสองตัว หรือลาก EA ตัวเดียวลงบนหลาย Chart พร้อมกัน จะทำให้ระบบสั่งงานซ้ำซ้อนและเกิดความผิดพลาดในการนับ P&L ครับ

Mobile Dashboard

Dashboard คือหน้าเว็บที่ใช้ดูสถิติและสั่งการ EA จากมือถือหรือคอมโดยไม่ต้องอยู่หน้าจอ MT5

ดูได้
  • Balance ปัจจุบัน
  • Daily Net P&L
  • Open P&L
  • จำนวน Cycles
  • Win / Loss
  • Symbol ที่กำลังเทรด
สั่งได้
  • SYSTEM ON — เปิดระบบ
  • SYSTEM OFF — ปิดระบบ
  • CLOSE ALL — ปิดไม้ทั้งหมด

เมื่อกดปุ่มบน Dashboard EA จะรับคำสั่งภายใน 3-5 วินาที (Poll ทุก 3 วิ) ไม่ต้องกดซ้ำครับ รอดูที่ HUD บน MT5 ว่าสถานะเปลี่ยนไหม

SECTION 08 — อ่านให้ครบ

คำเตือน สำคัญ

การลงทุนมีความเสี่ยง

การเทรด Forex และทองคำมีความเสี่ยงสูงมาก อาจขาดทุนได้ทั้งหมด ผลตอบแทนในอดีตไม่ได้รับประกันผลในอนาคต ลงทุนเฉพาะเงินที่รับความเสี่ยงได้เท่านั้น ไม่ควรนำเงินที่ใช้ในชีวิตประจำวันหรือเงินกู้มาลงทุนครับ

ช่วง News — ความเสี่ยงสูงสุด

ช่วงประกาศตัวเลขสำคัญเช่น Non-Farm Payrolls, CPI, Fed Rate Decision ทองอาจวิ่งแรงมากภายในวินาที EA อาจโดน Stop Loss หลายชุดติดกัน แนะนำ ตั้ง NewsSlot ใน EA เพื่อหยุดเทรดช่วง News หรือกด SYSTEM OFF ก่อนถึงเวลา News ครับ

License Key เป็นความลับ

License Key ผูกกับ Account Number ของคุณเท่านั้น ห้ามแชร์ให้ผู้อื่น ถ้า Key ถูกนำไปใช้กับบัญชีอื่น ระบบจะตรวจพบและระงับ Key ทันทีครับ

MT5 ต้องเปิดตลอดเวลา

EA ทำงานอยู่บนคอมของคุณ ถ้าคอมปิด ไฟดับ หรือ Internet หลุด EA จะหยุดทำงานทันที และออเดอร์ที่เปิดอยู่จะลอยค้างในตลาดโดยไม่มีการจัดการ แนะนำใช้ VPS ครับ

อย่าแก้ค่า Parameter ขณะเทรด

ห้ามแก้ค่า Lot, DailyTarget หรือ DailyMaxLoss ขณะที่มี Order เปิดอยู่ เพราะจะทำให้การคำนวณ P&L ผิดพลาด ถ้าต้องการเปลี่ยนค่าให้ ปิด Order ทั้งหมดก่อน แล้วจึง Remove EA และ Load ใหม่ครับ

ตลาด Sideways — EA ทำงานไม่ดี

ในช่วงตลาด Sideways หรือ Range ไม่มีทิศทาง EA อาจสัญญาณผิดบ่อย ถ้าเห็นว่าตลาดไม่มีทิศ แนะนำ กด SYSTEM OFF ชั่วคราว แล้วรอจนตลาดมีทิศทางชัดเจนก่อนครับ

EA ไม่ใช่ ATM

EA เป็นเครื่องมือช่วยเทรด ไม่ใช่การรับประกันกำไร บางวันตลาดไม่เอื้ออำนวย EA อาจทำกำไรได้น้อยหรือขาดทุนได้ ต้องเข้าใจว่า ขาดทุนเป็นส่วนหนึ่งของการเทรด ระบบ 5%/วัน คือการป้องกันไม่ให้ขาดทุนมากเกินไป ไม่ใช่ขาดทุนไม่ได้เลยครับ

ความเสี่ยง ระดับ วิธีป้องกัน
News Spike — ทองวิ่งแรงช่วง News สูงมาก ตั้ง NewsSlot หรือปิด EA ก่อน News

SECTION 10

คำถาม ที่พบบ่อย

EA ขึ้น CLOUD AUTH DENIED ทำยังไง?

ตรวจสอบ 3 อย่าง: 1) License Key ถูกต้องไหม 2) Account Number ตรงกับที่ลงทะเบียนไหม 3) อนุญาต WebRequest ใน MT5 แล้วหรือยัง

EA เปิดได้แต่ไม่เข้า Order?

ตรวจสอบว่า 1) กด SYSTEM ON แล้วหรือยัง 2) ชื่อ Indicator ใน InpIndiName ตรงกับไฟล์ที่ติดตั้งไหม 3) ไม่ได้อยู่ในช่วง NewsSlot หรือ Rollover ใช่ไหม

Dashboard บนมือถือเข้าไม่ได้?

ใช้ URL พร้อม Key ต่อท้ายเลยครับ เช่น https://...exec?key=PXXX-MMDD-RAND-SIGN เข้าได้ทั้งมือถือและคอมทุกเครื่อง

EA หยุดเองตอนกลางวัน เป็นอะไร?

ถึงเป้า DailyTarget หรือ DailyMaxLoss แล้วครับ HUD จะขึ้น DAILY GOAL REACHED ถือว่าปกติ ระบบจะ Reset ใหม่วันถัดไปครับ

อยากเพิ่มทุนและขยับ Lot ทำยังไง?

เมื่อพอร์ตเติบโตถึงระดับถัดไปในตาราง Lot ให้ 1) ปิด Order ทั้งหมดก่อน 2) Remove EA 3) Load EA ใหม่ 4) ตั้งค่า Lot และ DailyTarget ใหม่ตามตารางครับ
