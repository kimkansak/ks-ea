แผน 5 วันทดลอง

STEP BY STEP · DEMO ACCOUNT PROGRAM

DAY
1
ติดตั้งและเปิดใช้งาน

วันแรกคือการทำความรู้จักระบบ ติดตั้ง EA บนบัญชีเดโม่ ตั้งค่าพื้นฐาน และเปิดดู Dashboard ครั้งแรก

  • เปิดบัญชีเดโม่กับโบรกเกอร์ที่มี XAUUSDm
  • รับไฟล์ EA และ Indicator จาก KS Trader
  • ติดตั้งและลาก EA ลง Chart M5
  • เปิด Dashboard บนมือถือดูครั้งแรก
  • กด SYSTEM ON และดูสัญญาณแรก
🎯 เป้าหมาย — ระบบทำงานได้ AUTHORIZED
DAY
2
ดูสัญญาณและการเข้า Order

วันที่สองสังเกตว่า EA เข้า Order ตอนไหน ทำไม และ Dashboard แสดงผลอย่างไร

  • สังเกต Indicator บน Chart ว่าลูกศรขึ้นเมื่อไหร่
  • ดูว่า EA เข้า Order ตามสัญญาณจริงไหม
  • ดู P&L บน Dashboard ว่า Real-time ไหม
  • ทดสอบกด SYSTEM OFF แล้วดูว่า EA หยุดไหม
  • นับสัญญาณที่เกิดขึ้นในวันนี้
📊 เป้าหมาย — เข้าใจรูปแบบสัญญาณ M5
DAY
3
ทดสอบ Daily Target และ Stop

วันที่สามตั้งค่า DailyTarget และ DailyMaxLoss ตามทุนจริงที่วางแผนไว้ แล้วดูว่าระบบหยุดอัตโนมัติได้จริงไหม

  • ตั้ง DailyTarget = 10% ของทุนสมมติ
  • ตั้ง DailyMaxLoss = 5% ของทุนสมมติ
  • รอดูว่าถึงเป้าแล้วระบบหยุดไหม
  • ทดสอบกด EMERGENCY STOP ดูผล
  • บันทึก Win Rate ของวันนี้
🛡️ เป้าหมาย — ระบบ Auto Stop ทำงานได้
DAY
4
วิเคราะห์ผลและปรับ Parameter

วันที่สี่ดูภาพรวม 3 วันที่ผ่านมา ปรับ Lot และ Target ให้เหมาะกับสไตล์ของคุณ

  • ดู Win Rate รวม 3 วัน
  • ดูว่าช่วงเวลาไหน EA ทำกำไรได้ดีที่สุด
  • ลองเพิ่ม TotalLayers จาก 2 เป็น 3 ดูความต่าง
  • ลองปรับ TargetProfitUSD ดูผลกระทบ
  • สรุปว่าระบบเหมาะกับคุณไหม
🔧 เป้าหมาย — หาค่าที่เหมาะที่สุดสำหรับคุณ
DAY
5
ตัดสินใจและเริ่มบัญชีจริง

วันสุดท้ายสรุปผล 5 วัน ถ้าพอใจก็พร้อมเริ่มบัญชีจริงได้เลย ถ้ายังไม่มั่นใจ KS Trader พร้อมให้คำปรึกษาครับ

  • สรุปผลกำไร/ขาดทุนรวม 5 วัน (เดโม่)
  • คำนวณว่าถ้าใช้เงินจริงจะได้เท่าไหร่
  • ปรึกษา KS Trader เรื่องการตั้งค่าที่เหมาะสม
  • เลือกแพ็กเกจ Starter หรือ Premium
  • เปิดบัญชีจริงและเริ่มต้นได้เลย!
