1
ข้อสงวนสิทธิ์และความเสี่ยง

การเทรด Forex และทองคำ (XAU/USD) มีความเสี่ยงสูงมาก และอาจทำให้สูญเสียเงินลงทุนทั้งหมดหรือบางส่วนได้ KS ULTIMATE DATA EA เป็นเพียงเครื่องมือช่วยเทรด ไม่ใช่การรับประกันกำไร

  • ผลตอบแทนในอดีตไม่ได้รับประกันผลในอนาคต
  • ตัวเลขกำไรที่แสดงในสื่อโฆษณาเป็นการประมาณการเท่านั้น
  • ลงทุนเฉพาะเงินที่รับความเสี่ยงได้ ห้ามนำเงินที่ใช้ในชีวิตประจำวันหรือเงินกู้มาลงทุน
  • KS Trader ไม่รับผิดชอบต่อผลขาดทุนที่เกิดขึ้นจากการใช้ EA
2
สิทธิ์การใช้งาน License

License Key ที่ได้รับมีเงื่อนไขดังนี้:

  • 1 Key ต่อ 1 บัญชีเทรดเท่านั้น ไม่สามารถใช้กับหลายบัญชีพร้อมกัน
  • ห้ามแชร์ Key ให้ผู้อื่น ไม่ว่ากรณีใดก็ตาม ระบบจะตรวจพบและระงับ Key ทันที
  • ห้ามขาย โอน หรือให้เช่า Key ต่อโดยไม่ได้รับอนุญาต
  • Key มีอายุตามระยะเวลาที่ชำระเงิน เมื่อหมดอายุต้องต่ออายุก่อนใช้งานต่อ
  • KS Trader สงวนสิทธิ์ระงับ Key หากพบการละเมิดข้อตกลงนี้ โดยไม่คืนเงิน
3
ค่าบริการและการชำระเงิน
  • ค่าบริการรายเดือน Starter 3,500 บาท / Premium 5,500 บาท
  • ชำระเงินล่วงหน้าก่อนเริ่มใช้งาน ไม่มีการทดลองใช้งานบัญชีจริงฟรี
  • ทดลองใช้งานฟรี 5 วันบนบัญชีเดโม่เท่านั้น
  • ไม่มีการคืนเงิน หลังจากได้รับ License Key แล้วทุกกรณี ยกเว้นระบบมีปัญหาจาก KS Trader โดยตรง
  • ราคาอาจเปลี่ยนแปลงได้ KS Trader จะแจ้งล่วงหน้าอย่างน้อย 7 วัน
4
หน้าที่และความรับผิดชอบของผู้ใช้

ผู้ใช้งานมีหน้าที่รับผิดชอบดังนี้:

  • ตั้งค่า Lot และ Risk Management ด้วยตัวเอง ตามคำแนะนำในคู่มือ
  • ดูแลให้ MT5 และ Internet ทำงานต่อเนื่องตลอดเวลาที่ EA รันอยู่
  • ติดตามผลการเทรดด้วยตัวเอง อย่าปล่อยทิ้งโดยไม่ดูแลเป็นระยะ
  • ศึกษาคู่มือการใช้งานให้เข้าใจก่อนเริ่มใช้งานจริง
  • แจ้ง KS Trader ทันทีหากพบปัญหาผิดปกติของระบบ
  • รับผิดชอบผลลัพธ์จากการตั้งค่าที่ผิดพลาดด้วยตัวเอง
5
ข้อจำกัดความรับผิดชอบของ KS Trader

KS Trader ไม่รับผิดชอบ ต่อความเสียหายที่เกิดจาก:

  • การขาดทุนจากการเทรดไม่ว่ากรณีใดก็ตาม
  • ปัญหาจาก Broker เช่น Requote, Spread สูง, Server ล่ม
  • Internet หลุด คอมปิด หรือไฟดับระหว่างที่ EA ทำงาน
  • การตั้งค่า Lot หรือ Parameter ที่ผิดพลาดโดยผู้ใช้
  • ความเสียหายจากเหตุการณ์ที่ควบคุมไม่ได้ เช่น ข่าวด่วน วิกฤตตลาด
  • การใช้งาน EA ผิดวัตถุประสงค์หรือผิดจากคู่มือที่แนะนำ
6
การต่ออายุและการยกเลิก
  • Key จะหมดอายุอัตโนมัติตามวันที่กำหนด EA จะหยุดทำงานทันที
  • แจ้งต่ออายุก่อนหมด 3 วัน เพื่อไม่ให้การเทรดสะดุด
  • ถ้าไม่ต่ออายุภายใน 7 วันหลังหมด ข้อมูลใน Dashboard อาจถูกล้างครับ
  • ผู้ใช้สามารถยกเลิกได้ทุกเวลา แต่ไม่มีการคืนเงินส่วนที่เหลือ
  • KS Trader สงวนสิทธิ์ยกเลิก Key หากพบการละเมิดข้อตกลง
7
ความเป็นส่วนตัวของข้อมูล

  • ไม่มีการเข้าถึงรหัสผ่านบัญชีเทรดของคุณ ระบบไม่สามารถถอนเงินได้
  • ข้อมูลไม่ถูกเปิดเผยให้บุคคลที่สามโดยไม่ได้รับอนุญาต
  • ข้อมูลถูกเก็บบน Google Sheets ที่ปลอดภัยของ KS Trader
